Kanban Board Documentation

Kanban Board

A simple application to manage your Project and Task.

How to Use

Create a Project and Task. Tick the checkbox of the task. Drag the Project card to other columns depending on your situation. For example, if all your tasks are overdue you may move to Backlog. Or when all Tasks are done it moves to Done. You also can change the position by Edit Project → Status.

Project

Creating Projects

To create a Project, on the top right there is a button to create a Project. For projects, the required fields are:

  • Project Title
  • Priority Level
  • Status

Then click Save. The Project card will appear on the Status column.

Editing Projects

To edit, click at the title/header of the project card. The Project Details will appear and you can click Edit to change information if needed.

You also can add multiple tasks when creating a New Project or when editing a project, but it only asks for Task Name and Due Date.

Dates & Alerts

You also can add start and due dates to make alerts when the due date is near or the Project is overdue.

The appearance of the project card can be customized at Settings → Categories. Here you also can manage the list of Categories.

If you want to add a key person or who manages the project if needed, this can be managed at Settings → Key Person.

If you want to add more information you can put it in the Details section.

Task

The Task is linked to a project. So a project may have multiple tasks.

Creating Tasks

You can create a task by clicking the Task button on the top right home screen. The task requires:

  • Title
  • Project

Editing Tasks

To Edit a task, navigate to the project of the task card, click Info. For example, you create "Task 1" on "Example Project 1".

Adding Multiple Tasks

To add multiple tasks in a project you may go to Edit Project then add Tasks there, but it only adds task title and its due date.

Status Columns

By default there are: Backlog, Not Started, In Progress, Testing and Done. You can customize the arrangement, name and style at Settings → Kanban → Kanban Columns.

You can add or reduce status columns if needed depending on your usage.

Settings

Here are all settings for the Kanban Board app:

General

  • Document Name – The document name
  • Author – Author of the document
  • Version – Revision of the document
  • Theme – Change the app appearance
  • Font – Change the app font

Kanban

Change how many days before due date warning.

You also can enable Auto Delete tasks when all tasks in project are Done if needed.

Here you can add or remove Kanban columns as needed. Give it a name, color and optional icon for customization. The columns can be arranged except the first and last column that act as Start and Done.

Categories

Create and Edit Category name. Give it a name and pick a color. Your Project card will show the category name and it follows the color of the category.

The category can only be deleted if there are no projects associated with the category.

Key Persons

Manage the list of people who can be assigned to projects.

Sync

You can sync the data if you have a GitHub Account. Follow the steps and provide the details needed. Click Test Connection to check if the information is correct. If all OK, click Save Settings.

Manual Sync

You can manually save by clicking Push to Cloud, this will overwrite the file at GitHub Gist. You also can download your data to the app by clicking Pull.

Replace vs Merge

  • Replace – Will remove all current data and replace it with all data from GitHub Gist
  • Merge – Will only check if there is the same project, will update it and don't change any project that is not in GitHub Gist

Auto Save

There is also an option to Auto Save. The Save button on the main screen will blink green when autosave is running. But the autosave only sends when there are changes; if no change it will skip.

Clear Sync

To remove the Sync, just click Clear Sync Settings.

Data

Here you can export or import your backup of the Kanban. For import there are two modes: Replace and Merge.

You also can see the local storage usage. In general, browsers allocate 100KB to 5MB depending on settings and browser.

To clear the Kanban to default you can click Clear All Data. This will clear all data including the sync information.
